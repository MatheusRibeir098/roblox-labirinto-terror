# 🎃 Labirinto do Terror — Roblox

Jogo de terror no Roblox onde o jogador deve atravessar um labirinto infestado de monstros.

## Conceito

O jogador começa com uma arma e precisa chegar ao outro lado do labirinto.
No caminho, encontra monstros que pode **fugir**, **se esconder** ou **tentar matar**.

## Mecânicas planejadas

- [ ] Sistema de stamina (correr esgota energia)
- [ ] IA de monstros (patrulha, perseguição, campo de visão)
- [ ] Esconderijos funcionais (armários, embaixo de mesas)
- [ ] Geração procedural do labirinto
- [ ] Sistema de sanidade (distorção de tela perto de monstros)
- [ ] Modo cooperativo (2-4 jogadores)
- [ ] Sistema de progressão (skins, armas, dificuldades)
- [ ] Leaderboard

## Monetização

- **Game Passes** — modo difícil, personagem exclusivo, vida extra
- **Developer Products** — continuar após morte
- **Skins** — cosméticos de personagem e arma
- **VIP Server** — servidor privado com amigos
